# Marketing Funnel & Conversion Performance Analysis Using Python

## Project Overview

This project analyzes marketing funnel performance and customer conversion behavior using Python. The goal is to identify conversion trends, campaign effectiveness, customer engagement patterns, and opportunities to improve lead-to-customer conversion rates.

## Objectives

* Analyze customer responses to marketing campaigns.
* Measure overall conversion performance.
* Identify campaign acceptance trends.
* Evaluate customer segments based on education and marital status.
* Visualize funnel stages and conversion drop-offs.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Excel

## Dataset Features

The dataset contains customer demographic information, campaign responses, purchase history, and website engagement metrics.

Key columns:

* Education
* Marital_Status
* Income
* NumWebVisitsMonth
* NumWebPurchases
* AcceptedCmp1 – AcceptedCmp5
* Response

## Analysis Performed

### 1. Conversion Rate Analysis

Calculated the percentage of customers who responded positively to the marketing campaign.

### 2. Funnel Analysis

Analyzed the customer journey through:

* Website Visits
* Web Purchases
* Converted Customers

### 3. Campaign Performance

Compared acceptance rates across multiple marketing campaigns to identify the most effective campaigns.

### 4. Customer Segmentation

Evaluated conversion behavior based on:

* Education Level
* Marital Status

## Visualizations

* Marketing Funnel Chart
* Campaign Performance Bar Chart
* Education vs Conversion Rate
* Marital Status vs Conversion Rate

## Key Insights

* Measured overall customer conversion rate.
* Identified high-performing marketing campaigns.
* Analyzed customer engagement and purchase behavior.
* Highlighted customer segments with higher conversion potential.

## Project Outcome

The analysis provides actionable insights to improve marketing effectiveness, optimize campaign strategies, and increase customer conversion rates through data-driven decision making.
